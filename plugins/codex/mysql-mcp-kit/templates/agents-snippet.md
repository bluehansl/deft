## DB 접속·조회 (mysql MCP)

- DB 접속·조회가 필요하면 **반드시 `db-access` 스킬을 먼저 확인·사용한 뒤 진행**한다. 접속 구성(환경별 MCP 서버)·정책·자격증명·제약은 해당 스킬이 **단일 소스**로 소유한다.
- **핵심 backstop (스킬 미확인 시에도 준수)**: `write=true` 환경(쓰기)은 **반드시 사용자의 명시적 승인 후에만** 사용한다 — 승인 없이 쓰기 쿼리 실행 금지. (상세: `db-access` 스킬 §2)
